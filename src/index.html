Welcome to Eco-Sim!

This is an ecological simulator about bunnies living on a tropical island. The bunnies have only a few desires. They want to eat (carrots), drink (water), sleep and copulate from time to time.

Each bunny has genetic characteristics (such as speed and sight). Every newborn baby bunny inherits these traits from either the mother or the father. Genes additionally have the possibility to mutate.

You can follow a bunny by clicking on it. There are also statistics where you can track the success of the entire population.

To fasten up the simulation just increase the speed using the handle on the top right!

World
Tiles
The world consists of tiles. This value determines the number for each axis. The default value is 200 which would result in a 200 by 200 grid. Be aware that larger worlds require more processing power. [20-400]
Land/Water
The ratio between water and land. The higher the value, the more land mass is generated. [0-100]
Land Disruption
How much the land mass is disrupted by water. Kind of defines the size and amount of lakes. [0-100]
Tree Density
The amount of trees. [0-100]
Food Density
The amount of food. [0-100]
Bunnies
Genesis
The number of bunnies created by the FSM.
Mutation Chance
The chance that a gene mutation occurs. [0-100]
Mutation Amount
Specifies the maximum percentage of change when a mutation occurs. 0 = no change. Note that the resulting value is normally distributed (big changes are more unlikely). [0-100]
